

Browser Commands
     used to open the website, get page information, and close the page during automation testing

   1. page.goto()
           used to open the specified url

   2. page.title()
           used to get the title of page
           often used for verification whether correct page loaded

   3.page.url()
          used to get the current url of application
          useful for checking redirects or confirming navigation
   4. page.close()
          closes the active browser page which is opened by playwright
